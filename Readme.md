# MIST solution 高松支店 ホームページ

## アプリケーションの概要

MIST solution 高松支店でも、本社とは別でホームページを作成することになりました。

## アプリケーションの説明

- フレームワークにBootStrapを使う

## 動作環境の作り方

「site/index.html」をブラウザで開く

## 構成要素

- ホスト：Netlify

- CSSフレームワーク：BootStrap
